Introduction

Welcome to programming! With this course you will learn the basics of programming. You will learn and use the Python language syntax to exercise these programming basics. No prior programming skills or knowledge is needed for this course.

Why Python?

Why not?! Any programming language is suitable to start learning programming. As long as you want to learn and take your time. The Python language is being greatly used for teaching and learning programming. Therefore there are a ton of resources online that are usefull for learning.

What will you learn

As the course title says: The basics of programming. All these basics also translate to other programming languages. After successfully finishing this course you will be able to not only use Python in a very basic way, but also other programming languages after a little research on their syntax.

Subjects in this course are:

    Variables
    Datatypes
    Statements
    Operators
    Conditionals
    Functions
    Classes
    Methods
    Arrays

All subjects should be excercised chronologically for optimal understanding. Please contact one of your teachers whenever you are having trouble or a question. We would like to stimulate even small and seemingly stupid questions. Because there is no such thing as a stupid question!
